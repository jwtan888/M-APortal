How We Classify Laser Skill Level

Each completed laser row is tagged with a suggested complexity level. A PIC's Qualified Level is only shown when they have at least 3 completed rows at that level. Garment quantity is reference only and is not used for skill classification.

Level Classification Description
Level 1 Basic Process count <= 3 and one operation type. Small single-operation laser request.
Level 2 Intermediate Process count 4-8, or two operation types. Normal request with moderate handling.
Level 3 Advanced Process count 9-15 and two or more operation types. Multi-step laser handling with meaningful process count.
Level 4 Expert Process count 16-25 and two or more operation types. High process count requiring stable execution.
Level 5 Expert+ Process count > 25, all three operation types, and completed on or before required date.
💡

Calculation formula

  • Operation parts = laser parts + ironing parts + trim parts.
  • Actual minutes = complete date - start date.
  • SAM = actual minutes / operation parts.
  • Qualified level = highest level where PIC has at least 3 completed rows.
  • Skill percent = qualified level / 5.
  • Laser parts include laser cut and perforated parts. Trim parts means trim after bonding OT/BT.
